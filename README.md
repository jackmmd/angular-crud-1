#### Versiones
- Angular 12 
- NodeJs: v14.21.3
- Npm: 6.14.18

#### Instalar Angular
Comando para instalar angular 12 se debe ejecutar desde el command promt

```bash
npm i -g @angular/cli@12
```
Crear un proyecto
```bash
ng new my-app  
```
Crear una clase en la carpeta models
```bash
ng g cl models/employee 
```
